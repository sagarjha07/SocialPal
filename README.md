# SOCIALPAL (SOCIAL NETWORKING WEBSITE)

## A Social Networking website where user can:

- Login/Logout via signing up or using google oAuth
- Add a post/comment.
- Like a post/comment.
- Add a friend.
- Upload profile picture.
- ChatRoom to chat with everyone.
